Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM. The most common method is getElementById(), which selects an element using its unique ID. Another approach is getElementsByClassName(), which retrieves all elements with a specific class name. Similarly, getElementsByTagName() selects elements based on their tag names.
